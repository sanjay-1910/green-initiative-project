# Green Initiative Website

This project is a **responsive website** created to improve and practice my **HTML and CSS skills**.  
The website represents a **Green Initiative**, promoting environmental awareness and sustainable living.

The main goal of this project was to strengthen my understanding of **core front-end development concepts** such as layout design, responsive design, and modern CSS techniques.

---

## Technologies Used

- HTML5
- CSS3

---

## Concepts Practiced

In this project I practiced and implemented many important **HTML and CSS concepts**, including:

- Semantic HTML structure
- CSS Box Model
- Flexbox layout
- CSS Grid layout
- CSS Positions (static, relative, absolute, etc.)
- Media Queries
- Responsive Design
- Typography and styling
- Layout alignment and spacing

---

## Responsive Design

The website is fully responsive and adapts to different screen sizes using **CSS media queries**.

Screen Size

Description

**Less than 480px**

Mobile devices

**480px – 1024px**

Tablets

**Greater than 1024px**

Desktop screens

This ensures the website provides a good user experience across **mobile, tablet, and desktop devices**.

---

## How to Run the Project

To run the project locally:

1.  Download or clone the repository.
2.  Open the project folder.
3.  Open the file:

index.html

in your web browser.

You can also run it using **Live Server in VS Code** for a better development experience.
